# Music Game

Welcome to the Music Game!

This project involved developing a musical game to promote an approach to physiotherapy treatments focused on upper motor movements. The goal is to increase patient engagement during sessions and improve their adherence to rehabilitation appointments.

Developed on the Unity platform, the code can be adapted to meet the specific needs of each user, ensuring a personalized experience.

## Step by step for installation and execution:
- The file "sharedassets3.assets.rar" needs to be extracted to the Music_Game_Data folder.

## Highlights of the "Music" Game

The game "Music" was designed to be played with an external controller, providing an immersive and interactive experience.

# Music Game
 
Bem-vindo ao Jogo Music! 

Neste projeto foi desenvolvido um jogo musical para promover uma abordagem aos tratamentos de fisioterapia com foco nos movimentos motores superiores. O objetivo é aumentar o envolvimento do paciente durante as sessões e melhorar sua adesão às consultas de reabilitação.

Desenvolvido na plataforma Unity, o código pode ser adaptado para atender às necessidades específicas de cada usuário, garantindo uma experiência personalizada.
